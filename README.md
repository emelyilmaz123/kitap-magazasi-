# 📚 Kitap Mağazası

Tam stack kitap satış uygulaması.

## Teknolojiler

**Frontend**
- React
- React Router DOM

**Backend**
- Node.js
- Express

**Veritabanı**
- PostgreSQL (Supabase)
- Prisma ORM

## Kurulum

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Özellikler
- Kitap listeleme ve kategori filtreleme
- Kullanıcı giriş / kayıt (JWT)
- Kitap detay sayfası
- Sepet yönetimi
- Admin paneli
